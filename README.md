# X Articles Converter

Markdown to X Articles (formerly Twitter Articles) formatting tool. Build with a single HTML file, no build process required.

## Features

- **Markdown Support**: Headers, lists, bold, italic, links, blockquotes, code blocks.
- **Table Conversion**: Automatically converts Markdown tables to list format (since X Articles doesn't support HTML tables).
- **Image Stripping**: Automatically removes markdown image syntax (images must be uploaded manually in X Articles editor).
- **Theme Switching**: Dark (default X style) and Light modes.
- **Responsive Design**: Works on desktop and mobile.
- **Clipboard Copy**: One-click copy with preserved formatting for X Articles.

## Usage

1. **Write**: Paste your Markdown content into the left editor.
2. **Preview**: See the real-time preview on the right.
3. **Copy**: Click "Copy to X Articles".
4. **Paste**: Go to X Articles editor and paste (Cmd+V / Ctrl+V).

## Local Development

Simply open `index.html` in your browser.

```bash
open index.html
```
